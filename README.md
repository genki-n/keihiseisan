# Receipt History Downloader (Chrome Extension / Manifest V3)

Amazon.co.jp と Yahoo!ショッピングの領収書PDFを解析し、`YYYYMMDD_仕入先_金額円_領収書.pdf` 形式で Downloads 配下に保存する拡張機能です。

## 1. Chrome拡張の読み込み方法

1. `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このフォルダを選択する

## 2. Amazon注文履歴での使い方

1. Amazon注文履歴ページを開く（ログイン状態）
2. 拡張アイコンをクリック
3. サービスで Amazon または 両方 を選択
4. 年月を選択
5. 「この月の領収書を保存」を押す
6. ログ欄で成功/失敗を確認

## 3. Yahoo!ショッピングでの使い方

1. Yahoo!ショッピングの注文履歴ページを開く（ログイン状態）
2. 拡張を開く
3. サービスで Yahoo!ショッピング または 両方 を選択
4. 年月を選択
5. 実行

## 4. 手動PDFリネーム機能の使い方（優先機能）

1. 拡張ポップアップを開く
2. 「PDFファイルを選択」で複数PDFを選択
3. 「選択したPDFをリネームして保存」を押す
4. PDF本文を解析して、命名規則に従って保存

## 5. 保存先

- Amazon: `Downloads/領収書/Amazon/`
- Yahoo: `Downloads/領収書/Yahoo/`

保存ファイル例:
- `領収書/Amazon/20260302_Amazon_2016円_領収書.pdf`
- `領収書/Yahoo/20260306_ヤフーショッピング_2940円_領収書.pdf`

同名ファイルがある場合は `uniquify` で重複回避します。

## 6. 注意事項

- Amazon/Yahooのページ仕様変更で自動取得が失敗する可能性があります
- 失敗時は、PDFを一括ダウンロードして手動PDFリネーム機能を使用してください
- ログイン状態が必要です
- CAPTCHA/本人確認が出た場合は手動対応が必要です
- このリポジトリでは `lib/pdf.min.js` をローカル同梱していますが、オフライン環境では簡易抽出フォールバックを使用します

## 解析ルール

- Amazon: `注文日 YYYY-MM-DD`、`合計 ￥x,xxx` を抽出
- Yahoo: `注文日: YYYY年MM月DD日`、`合計金額: x,xxx円` を抽出
- 日付は `注文日` 優先（発行日ではない）
